# 🤖 System Guardian - One-Click Launcher

Welcome to the **System Guardian Onboarding Bot**. You can now launch the entire AI Security platform directly from this folder.

### 🚀 How to Run
- **Windows User**: Simply double-click `START_GUARDIAN.bat`.
- **Python Expert**: Run `python RUN_PROJECT_BOT.py`.

### 🛠️ What the Bot Does
1. **Initializes**: Sets up the working directory automatically.
2. **Cleans**: Stops any conflicting background processes.
3. **Deploys**: Boots the AI Core on port 5006.
4. **Onboards**: Automatically opens your browser to the Intelligence Dashboard.

### 📊 Project Structure
- `START_GUARDIAN.bat`: The primary one-click launcher.
- `RUN_PROJECT_BOT.py`: Advanced Python-based launch sequence.
- `system_guardian/`: The core engine and dashboard files.
